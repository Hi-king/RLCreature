# RLCreature
OSS版人工生命

サンプルデモを動かす
===========================

0. Unityを用意

- 2018が推奨
- ScriptingRuntimeVersionを必ず.NET4.Xにして動かして下さい

1. レポジトリをクローン

- git-lfsが必要です

2. 依存ライブラリをインストール

- MessagePack-CSharp
  - https://github.com/neuecc/MessagePack-CSharp/releases
    - unitypackageをダウンロードしてインストールしてください

3. デモシーンを開く

- ひたすらキューブが進むだけ
  - Assets/Sample/SimpleHunting
